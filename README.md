# Clay, Gesture and Stigmergic Processes

This repository documents the material and process-based aspects of a research project exploring stigmergy through clay practices.

While stigmergy is often modeled through stable traces in computational and biological systems, this work investigates how coordination emerges under conditions of instability, error and material precarity.

The project includes experimental processes, installation development and performative interactions with clay, where gesture and material response continuously reshape each other.

This repository focuses on material experimentation and complements a theoretical dissertation and computational models developed in parallel.
